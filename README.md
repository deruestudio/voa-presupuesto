# Propuesta Producción Gráfica VOA

Archivo listo para subir a GitHub Pages.

## Uso
1. Crear un repositorio en GitHub.
2. Subir `index.html` a la raíz del repositorio.
3. Activar GitHub Pages desde Settings > Pages.
4. Usar la URL pública en Carrd como iframe.

El logo DERU está incrustado dentro del HTML, no requiere archivos externos.
